# 🪟 Operating Systems

Operating systems are the foundation of modern computing. Understanding how they work is essential for both Information Technology and Cybersecurity, as they provide the environment where users, applications, services, and networks interact.

This section documents the operating system concepts I've studied, the practical skills I've developed, and the hands-on experience I've gained while learning Windows and Linux. Everything included here reflects my personal learning journey through certification studies, labs, technical documentation, and continued practice.

---

# 📚 Areas of Study

### 🪟 Windows

Learning Windows administration, system configuration, user management, file systems, permissions, command-line tools, and troubleshooting.

---

### 🐧 Linux

Building a strong understanding of Linux fundamentals including the command line, file systems, permissions, package management, users, groups, and basic system administration.

---

### 💻 Command Line

Developing confidence using Windows Command Prompt, PowerShell, and the Linux terminal to navigate systems, manage files, troubleshoot issues, and perform common administrative tasks.

---

### 📁 File Systems & Permissions

Learning how operating systems organize data, manage storage, and control access through users, groups, ownership, and permissions.

---

### 🛠️ System Troubleshooting

Practicing structured troubleshooting techniques to identify, diagnose, and resolve common operating system issues while building confidence through hands-on experience.

---

# 💡 Learning Philosophy

Operating systems are more than software. They're the foundation that connects hardware, networking, users, and applications. The better I understand how they work, the better prepared I am to troubleshoot problems, support users, and secure the systems they rely on.

This section isn't intended to be a complete reference for Windows or Linux. Instead, it documents the knowledge I've personally built as I continue strengthening my Information Technology foundation.
