# 🔎 System & Network Discovery

System and network discovery focuses on identifying devices, services, ports, and other information that helps build an understanding of an environment. By gathering this information, security professionals can better assess network visibility, support investigations, and identify potential security risks.

This section documents the system and network discovery concepts I've studied, the practical skills I'm developing, and the hands-on experience I'm gaining through certification studies, labs, and continued practice. Everything included here reflects my personal learning journey as I continue building the knowledge needed for a career in cybersecurity.

---

# 📚 Areas of Study

### 🖥️ Host Discovery

Learning how discovery tools identify active devices within a network and provide visibility into connected systems.

---

### 🔌 Port & Service Identification

Building an understanding of how open ports and running services provide insight into how systems communicate and what services are available.

---

### 🗺️ Network Visibility

Learning how discovery tools help security professionals develop a clearer understanding of network environments and support security investigations.

---

### 📊 Information Gathering

Developing the ability to collect and interpret discovery results while understanding how they contribute to troubleshooting, security assessments, and defensive planning.

---

### 📚 Continuous Learning

Strengthening my understanding of system and network discovery through certification studies, hands-on labs, technical documentation, and practical exercises while continuing to expand my cybersecurity knowledge.

---

# 💡 Learning Philosophy

Discovery tools help answer one of the most important questions in cybersecurity: *What exists within the environment?* Before systems can be monitored, protected, or investigated, they first need to be understood. As I continue learning, I focus on building the knowledge needed to interpret discovery results rather than relying solely on the tools that produce them.

This section isn't intended to be a complete reference for system and network discovery. Instead, it documents the knowledge I'm personally building as I continue working toward a career in cybersecurity.
